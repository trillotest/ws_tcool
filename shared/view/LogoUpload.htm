This interface is provided here as a convenience to change the default logo image easily. Any image loaded from here is used to create three more images as follows. All images are stored under "shared>Images" menu.
  1. logoStamp.[png, gif, jpg] : a proportionally sized image with maximum height of 33 px or maximum width 60 px. The extension is same as of the original image.
  2. logoSmall.[png, gif, jpg] : a proportionally sized image with maximum dimension 75x75 px.
  3. logoRegular.[png, gif, jpg] : a proportionally sized image with maximum dimension 100x200 px.
  4. original image : the original image is also saved under shared>Images.
If an image with the matching name is found, the old image is renamed to a unique name constructed using "filename(n).ext" format. By default applications use "logoStamp.[png|gif|jpg]" image. Later you can customize it to anything by editing HTML or CSS. You can also load images using Image menu under Dev environment and use them in logos.
